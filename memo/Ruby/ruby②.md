## 演算子

+  足し算

-  引き算

*  掛け算

/  割り算

%  割り算の余りを計算

** べき乗

#### 比較
<  右が大きい

<= 右が大きいまたは等しい

>  左が大きい

>= 左が大きいまたは等しい

== 等しい

!= 等しくない

#### 論理演算子
!  否定

&& 左右どちらも正しければ真

|| 左右どちらかが正しければ真


#### 変数の数値の増減

+=  値を増加

-=  値を減少

*=  値を倍増

/=  値を割る

**= 値をn乗


## 数値と文字列の型変換
#### 数値の文字列を計算する際は数値に変換しなければならない
#### 変数に数値を代入して文字列と連結される際は数値を文字列に変換しなければならない

    1 + '10'  誤
#### to_iメソッドを使う

    1 + '10'.to_i  正  文字列型→整数型
=> 11

#### to_sメソッド

    number = 5

    'number is' + number  誤
    'number is' + number.to_s  正  整数型→文字列型
=> number is 3

### 式展開
#### 式展開を使用する場合はto_sメソッドは必要ない

    "Number is #{number}"
=> "Number is 3"
